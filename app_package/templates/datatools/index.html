Data Tools

  • File Share Application

    This application is a means to share files through the internet. It functions very similar to Dropbox or Google Drive. Users can upload and download small or very large files to a safe place on the internet.

  • NHTSA Recalls and Investigations Dashboard

    National Highway Traffic Safety Administration (NHTSA) lists all recalls and investigation of US automotive manufactures vehicles. This application stores all records in a database and allows users a front end to add notes, files, link records and add/remove records.
    This is the development server application used for Kinetic Metrics. There is no proprietary data.

  • Shiftgarten Management Dashboard

    The first of my dash and database products. Click here (and Login as guest) to preview a restaurant management application. Data is all mock data. Feel free to poke around.
    This is the development server application used for Creative Bar Concepts. There is no proprietary data.

  • CAGE Code Lookup

    Using the Defense Logistic Agency's batch text file I have stored the searchable information on this site.

  • BLS Indices

    Download the latest Bureau of Labor (BLS) producer price indices (PPI) in a formatted excel file

  • What Sticks Health

    What-Sticks-Health is a health data storage platform that provides feedback to the user. The goal of What-Sticks is unapologetically crude. We are throwing the spaghetti plate of data against the wall and seeing what sticks. Currently, it allows for Polar Personal device uploads... more coming soon!

  • Security

    Retrieve HTTP Strict-Transport-Security (HSTS) policy from URLs. Type in URLs or upload a csv with URLs to search and return a table or csv with HSTS policies.
